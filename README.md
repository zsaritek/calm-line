# Calm-Line

Calm-Line is a small SwiftUI app that shows one quote a day.
Nothing more, nothing noisy.

You open the app, read a short quote, and move on with your day.

---

## What it does

- Shows a daily quote that stays the same throughout the day
- Lets you refresh and read a different quote if you want
- Save quotes to your favorites
- Share quotes easily
- Simple settings (haptics, author visibility)

---

## Why this app

Calm-Line was built as a lightweight SwiftUI project focused on:
- clean UI
- simple state management
- readable code
- finishing something small, properly

No over-engineering, no extra dependencies.

---

## Tech stack

- Swift 5.9
- SwiftUI (iOS 17+)
- MVVM (lightweight)
- UserDefaults for persistence

---

## Project structure

- Models: quote data
- Services: daily quote logic, favorites storage
- ViewModels: UI state
- Views: small, focused SwiftUI views

Kept intentionally simple.

---

## How to run

1. Clone the repository
2. Open `Calm-Line.xcodeproj`
3. Run on iOS Simulator or device (iOS 17+)
